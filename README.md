# Introduction

This is a shell script that allows you to delete a file and move it to a temp folder at the home directory.

## Commands

* To delete a file type

```smartdel -d "File Name"```
or
```smartdel "File Name"```

* To recover a deleted file, type

```smartdel -r "File name"```
